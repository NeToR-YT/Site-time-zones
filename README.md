# Часові пояси — простий сайт

Цей невеликий статичний сайт показує поточний час у часовому поясі користувача та дозволяє додавати інші часові пояси для спостереження. Вибрані часові пояси зберігаються в `localStorage` браузера.

Як використовувати
- Відкрийте `index.html` в браузері (подвійний клік або через "Open File" в браузері/IDE).
- Ваш часовий пояс визначається автоматично.
- Використайте поле пошуку, виберіть часовий пояс у списку та натисніть "Додати" або двічі клікніть рядок в списку.
- Щоб прибрати картку, натисніть × на ній. Щоб очистити всі — натисніть "Очистити".

Зауваги
- Список часових поясів на сучасних браузерах отримується через `Intl.supportedValuesOf('timeZone')`. Якщо ця функція недоступна, використовується запасний короткий список найбільш вживаних часових поясів.
- Це статична сторінка — сервер не потрібен.

Можливі покращення
- Показати секунди/аналоговий годинник, додати сортування за відстанню від користувача, додати міжнародну локалізацію інтерфейсу.

Автор: автоматичний шаблон
